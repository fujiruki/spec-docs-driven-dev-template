## 書き方ガイド

雑でいい。思いついたことをそのまま書く。

```markdown
- ログイン機能にGoogleアカウント対応を追加してほしい
- ダッシュボードに月次売上グラフを表示したい
```

優先度をつけたい場合:
- **【高】** 業務に支障がある / バグ / 納期が迫っている
- **【中】** あると便利 / 改善要望
- **【低】** いつかやりたい / アイデア段階
